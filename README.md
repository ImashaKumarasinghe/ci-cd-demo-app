# Full CI/CD Pipeline for Full-Stack Web Application

## Overview
This project demonstrates a full-stack web application with an automated CI/CD pipeline.

## Tech Stack
- React + Vite
- Node.js
- Express.js
- MongoDB Atlas
- GitHub Actions
- Render / Vercel

## Features
- Add items
- View items
- Delete items
- Frontend and backend integration
- MongoDB database connection
- Automated CI pipeline
- Auto deployment after GitHub push

## CI/CD Workflow
Code push → GitHub Actions checks → Render/Vercel auto deploy

## GitHub Actions Pipeline Includes
- Backend dependency installation
- Backend tests
- Backend security audit
- Frontend dependency installation
- Frontend production build
- Frontend security audit
